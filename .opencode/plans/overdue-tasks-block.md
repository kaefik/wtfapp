# Блок просроченных задач на странице Today

## Цель

Показывать невыполненные задачи с просроченным дедлайном отдельным свёрнутым аккордеоном вверху страницы Today.

## Контекст

Сейчас страница Today (`DueDateTaskList` с `dayOffset=0`) показывает только задачи с дедлайном на сегодня. Просроченные задачи (due_date < сегодня) нигде не выделены в отдельную группу — они доступны только в общем списке с красной подсветкой даты. Пользователь может не заметить просроченные задачи.

## Дизайн

### 1. Хук `useOverdueTasks`

**Файл:** `frontend/src/hooks/useOverdueTasks.ts`

Отдельный хук, запрашивающий просроченные задачи из Dexie:
- `useLiveQuery` к `db.tasks`
- Фильтры: `!isCompleted`, `!trashedAt`, `dueDate !== null`, `dueDate < startOfDay(today)` в таймзоне пользователя
- Переиспользует утилиту `getDayBounds` из существующего кода
- Возвращает массив задач, отсортированных по `position`
- Принимает `timezone: string` как параметр

### 2. Компонент `OverdueTasksBlock`

**Файл:** `frontend/src/components/OverdueTasksBlock.tsx`

Аккордеон-компонент:
- Свёрнут по умолчанию
- Заголовок: «Просрочено (N)» где N — количество задач
- По клику раскрывается с анимацией
- Внутри — список задач через существующий `TaskListView`
- Не рендерится если просроченных задач нет (N=0)
- Стили: акцентный цвет заголовка, иконка предупреждения

### 3. Интеграция в `DueDateTaskList`

**Файл:** `frontend/src/routes/DueDateTaskList.tsx`

- Импортирует `useOverdueTasks` и `OverdueTasksBlock`
- Вызывает `useOverdueTasks(timezone)` при `dayOffset === 0`
- Рендерит `OverdueTasksBlock` перед основным списком задач
- При `dayOffset !== 0` блок просрочки не отображается

## Что не меняется

- Backend не трогается
- Модель данных не меняется
- Страница Tomorrow не затрагивается
- Существующая подсветка красным в `TaskListView` сохраняется

## Критерии успеха

- Просроченные задачи видны на странице Today в свёрнутом аккордеоне
- Аккордеон раскрывается/сворачивается по клику
- Не отображается если просроченных задач нет
- Tomorrow-страница работает без изменений
